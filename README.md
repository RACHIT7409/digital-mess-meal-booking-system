# Digital Mess Meal Booking System

A MERN stack project for hostel mess meal booking, payment tracking, QR coupon verification, refund handling, and admin/manager dashboards.

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS
- JWT Authentication
- QR Code Verification

## Features

- Student registration/login
- Meal booking
- Dummy payment
- QR coupon generation
- Manager QR verification
- Mark meal as served
- Mark booking as not served
- Refund request and approval
- Admin user management
- Meal management
- Reports and dashboard

## Backend Setup

```bash
cd backend
npm install
cp .env.example .env
npm run dev
